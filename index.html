We aim to develop a website that makes use of Information Retrieval to find the best-suited article according to the search query given by the user. We plan to rank the articles on the web using the BM25 algorithm. In information retrieval, BM25 is a ranking function used by search engines to estimate the relevance of documents to a given search query.
